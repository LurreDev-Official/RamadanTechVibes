# Ramadan Goals Tracker

Track your spiritual goals and habits during Ramadan with an easy-to-use progress tracker.

## Features
- Add custom goals with categories (Quran, Prayer, Charity, etc.)
- Set target numbers for each goal
- Track progress with visual progress bars
- View completion statistics
- Filter goals by status (All, Active, Completed)
- Reset or delete goals
- Data persists in browser localStorage

## Usage
1. Add a new goal with title, category, and target
2. Click "+1 Progress" to increment your progress
3. Watch your completion rate increase
4. Filter goals to see active or completed ones
5. Reset or delete goals as needed

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage for data persistence
