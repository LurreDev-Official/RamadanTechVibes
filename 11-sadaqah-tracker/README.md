# Sadaqah Tracker

Track and manage your charity donations (Sadaqah) with a simple, beautiful interface.

## Features
- Add donation records with amount, cause, date, and notes
- View total amount donated and number of donations
- Filter donations by time period (All, This Month, This Year)
- Delete donations
- Data persists in browser localStorage
- Beautiful statistics dashboard

## Usage
1. Fill in the donation details (amount, cause, date, notes)
2. Click "Add Donation" to record it
3. View your donation history and statistics
4. Use filters to view donations by time period
5. Delete donations if needed

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage for data persistence
