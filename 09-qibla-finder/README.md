# Qibla Finder

Find the direction to the Kaaba in Makkah from anywhere in the world.

## Features
- Visual compass showing Qibla direction
- Automatic location detection
- Manual coordinate input
- Shows exact angle to Qibla
- Displays distance to Kaaba in kilometers
- Real-time calculations using geolocation

## Usage
1. Click "Use My Location" to automatically detect your location
2. Or enter your coordinates manually (latitude and longitude)
3. The compass will show the Qibla direction with a red arrow
4. View the exact angle and distance to the Kaaba

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- Geolocation API
- Haversine formula for distance calculation
