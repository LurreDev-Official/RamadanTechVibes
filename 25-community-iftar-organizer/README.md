Organize and manage community iftar events during Ramadan.

## Features

- Create new iftar events
- View upcoming and past events
- Event details (date, time, location, attendees)
- Delete events
- LocalStorage persistence
- Responsive design

## Technologies Used

- HTML5
- CSS3 with grid layout and gradients
- Vanilla JavaScript
- LocalStorage API

## How to Use

1. Open `index.html` in a web browser
2. Fill in the event form with details
3. Click "Create Event" to add the event
4. View all events in the list below
5. Click on any event to see full details
6. Delete events if needed
