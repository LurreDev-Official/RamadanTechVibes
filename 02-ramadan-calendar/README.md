Interactive Ramadan calendar to track your fasting days.

## Features
- Visual calendar showing all 30 days of Ramadan
- Click to mark days as completed
- Local storage to save your progress
- Responsive design for mobile and desktop

## How to Use
1. Open `index.html` in your browser
2. Click on any day to mark it as completed (fasted)
3. Click again to unmark
4. Your progress is automatically saved

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API
