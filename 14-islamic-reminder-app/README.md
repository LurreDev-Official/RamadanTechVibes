# Islamic Reminder App

Set reminders for prayers, Quran reading, and other Islamic activities with desktop notifications.

## Features
- Set custom reminders with title, time, and type
- Daily repeating or one-time reminders
- Desktop notifications when reminder time is reached
- Toggle reminders active/inactive
- View current time
- Data persists in browser localStorage
- Automatic reminder checking every minute

## Usage
1. Click "Enable Notifications" to allow browser notifications
2. Add a reminder with title, time, type, and repetition option
3. Reminders will automatically trigger at the set time
4. Toggle reminders on/off or delete them as needed

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- Web Notifications API
- LocalStorage for data persistence
- setInterval for time checking
