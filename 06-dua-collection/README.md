# Dua Collection

A beautiful collection of daily Islamic duas with Arabic text, transliteration, and English translation.

## Features
- 10+ essential duas for daily life
- Category filtering (Morning, Evening, Daily, Special)
- Arabic text with proper formatting
- Transliteration and translation
- Clean, modern UI

## Usage
Simply open `index.html` in your browser and browse through the duas. Use the category filter to find specific types of duas.

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
