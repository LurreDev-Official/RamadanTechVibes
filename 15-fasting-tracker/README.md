# Fasting Tracker

Track your fasting days, view statistics, and maintain your fasting streaks.

## Features
- Track fasting days with types (Obligatory, Voluntary, Makeup, etc.)
- View comprehensive statistics (total fasts, current streak, longest streak)
- Interactive calendar showing fasted days
- Quick toggle for today's fast
- Add past fasting days with notes
- View recent fasting history
- Calculate and display streaks automatically
- Data persists in browser localStorage

## Usage
1. Click "Mark as Fasted" to record today's fast
2. Or add past fasting days using the form with date and type
3. View your statistics at the top
4. Browse the calendar to see all fasted days
5. Check your recent fasting history at the bottom

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage for data persistence
- Dynamic calendar generation
- Streak calculation algorithm
