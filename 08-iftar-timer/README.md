# Iftar Timer

Countdown timer to help you prepare for iftar (breaking fast) at Maghrib time.

## Features
- Real-time countdown to iftar
- Set custom Maghrib time
- Quick time presets
- Visual countdown display
- Desktop notifications (when permitted)
- Shows current time
- Automatic daily reset

## Usage
1. Set your Maghrib (iftar) time using the time picker or quick buttons
2. Click "Start Countdown" to begin
3. Watch the countdown to iftar time
4. Get notified when it's time to break your fast

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- Web Notifications API
