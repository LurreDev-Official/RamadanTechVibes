# Hijri Date Converter

Convert dates between Gregorian (Western) and Hijri (Islamic) calendars with accurate calculations.

## Features
- Gregorian to Hijri conversion
- Hijri to Gregorian conversion
- Display today's date in both calendars
- Accurate Julian day number calculations
- Clean, intuitive interface

## Usage
1. **Gregorian to Hijri**: Select a Gregorian date and click convert
2. **Hijri to Gregorian**: Enter day, month, and year in Hijri calendar and click convert
3. View today's date in both calendars at the bottom

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- Julian Day Number algorithm for accurate conversions
