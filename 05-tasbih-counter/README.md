Digital dhikr counter for your daily remembrance of Allah.

## Features
- Simple click counter
- Progress bar visualization
- Preset targets (33, 99, 100)
- Auto-rotating dhikr phrases (Subhanallah, Alhamdulillah, Allahu Akbar)
- Keyboard support (Spacebar to count)
- Vibration feedback on mobile

## How to Use
1. Open `index.html` in your browser
2. Click the + button or press Spacebar to count
3. Select preset targets for common dhikr counts
4. Click Reset to start over
