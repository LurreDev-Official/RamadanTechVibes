# Islamic Quotes Generator

Get inspired with beautiful Islamic quotes from the Quran, Hadith, and Islamic wisdom.

## Features
- 15+ inspiring quotes
- Category filtering (Quran, Hadith, Wisdom)
- Random quote generation
- Share quotes
- Copy to clipboard
- Save favorite quotes (persists in browser)
- Beautiful gradient design

## Usage
1. Click "Generate New Quote" to get a random quote
2. Filter by category using the filter buttons
3. Add quotes to favorites for later reference
4. Share or copy quotes to clipboard

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage for favorites persistence
- Web Share API
