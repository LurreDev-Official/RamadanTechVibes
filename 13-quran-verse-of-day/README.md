# Quran Verse of the Day

Display beautiful verses from the Holy Quran with Arabic text and English translation.

## Features
- 15+ beautiful Quran verses
- Arabic text with English translation
- Random verse generation
- View history of recent verses (last 5)
- Share verses
- Copy to clipboard
- Beautiful gradient design with Islamic styling

## Usage
1. Click "Get New Verse" to display a random Quran verse
2. View the Arabic text, reference, and English translation
3. Share or copy verses to clipboard
4. Click on recent verses in the history section to view them again

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage for history persistence
- Web Share API
